# Aplicação para para coleta e envio de dados via Bluetooth

## Instalar dependencias
Para rodar o projeto primeiramente é necessario instalar as dependencias, então rode o seguinte comando
```bash
npm install
```
## Rodar o projeto
Para executar o projeto é necessario dentro da pasta do projeto rodar o comando
```bash
npx expo start --dev-client
```
